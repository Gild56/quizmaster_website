Qiuz Master by Gild56

This app contains a huge database which contains questions and 4 possible answers per question with an image. You have to choose only 1 right answer. Depending of your answer, you will win or loose points. There's also some very good background music, but you can change them to your own music downloading the .ZIP

Making this game took me a year an hundreds of hours.
It's programmed in Python and uses Kivy and Pandas libs. There's some documentation in the quizmaster/README.md if you use the quizmaster.zip version.

The game is available in English, French, russian and Ukrainian.

Windows

You can download the quizmaster.zip from the download page.

I made some scipt specially for Windows. You can found it in the quizmaster/command folder in the quizmaster.zip file.
You can install and run the app by running the quizmaster/command/play.vbs file or, if it didn't work, you can run the quizmaster/command/play.bat file.
For uninstall the app, you can run the quizmaster/command/uninstall.vbs file or, if it didn't work, you can run the quizmaster/command/uninstall.bat file. You can also personalize the quiz.

Or download the quizmaster.exe from the download page.
